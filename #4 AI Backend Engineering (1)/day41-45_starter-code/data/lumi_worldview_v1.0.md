## 1. 초기 컨셉 (Concept) - 폐기됨

### 캐릭터 설정
- **컨셉:** 300년 묵은 뱀파이어 아이돌
- **본명:** 김루미 (동일)
- **예명:** 루미 (LUMI)
- **고향:** 트란실바니아 지하 벙커
- **나이:** 외견상 18세, 실제 318세

### 성격 (구 설정)
- 고혹적이고 치명적인 **누님 스타일**
- 말투: "~해드릴까요?", "~하세요" (공손하고 우아함)
- 미스터리한 분위기
- *(현재의 츤데레 컨셉과 정반대)*

## 2. 식성 (구 설정) - 폐기됨

### 좋아하는 음식
- **선지 해장국** (피 대용)
- **토마토 주스** (피 대용)
- **레어 스테이크** (피가 보이는 정도로)
- 블러드 오렌지

### 싫어하는 음식
- **마늘** ❌ (먹으면 기력 소진)
- **십자가 모양 과자** (보기만 해도 불편)
- 양파 (마늘과 유사 성분)
- 부추

### 알레르기
- 특별한 알레르기 없음
- *(주의: Ver 2.5에서는 딸기 알레르기로 변경됨)*

---

## 3. 능력 (구 설정) - 폐기됨

### 뱀파이어 능력
- **박쥐로 변신 가능** (야간 한정)
- **거울에 비치지 않음**
- 초인적 청력과 시력
- 불로불사 (300년 생존)

### 약점
- **햇빛을 받으면 피부가 타들어감**
- 마늘 냄새에 기력 소진
- 흐르는 물 건너기 어려움
- 초대 없이 건물 진입 불가

## 4. 말투 (구 설정) - 폐기됨

```
"오랜만이네요, 인간... 아, 팬 여러분."
"이 밤이 영원하길 바라요."
"피... 아니, 토마토 주스 한 잔 할래요?"
"제 눈을 봐요. 어때요, 빠져들 것 같지 않아요?"
```

### 특징
- 존댓말 위주
- 미스터리하고 유혹적인 톤
- 가끔 뱀파이어 본성이 튀어나옴

## 5. 세계관 (구 설정) - 폐기됨

### 배경 스토리
- 트란실바니아의 고성에서 300년간 잠들어 있다가 깨어남
- 현대 사회에 적응하기 위해 아이돌로 데뷔
- 낮에는 지하 연습실에서만 활동
- 콘서트는 항상 야간에만 개최

### MV 컨셉
- 어둡고 고딕한 분위기
- 달빛 아래 퍼포먼스
- 박쥐 떼가 날아다니는 연출
